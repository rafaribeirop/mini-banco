## Mini Bank 💌

A simple Java console project that simulates a small bank system
## Features 
- Create an account (Current or Savings)
- Deposit money 
- Withdraw money 
- Pay monthly fees  (different for Current and Savings accounts)
- Check account status 
- Close account (only if the balance is zero)

## How It Works 
- You type your first name 
- Choose the type of account: Current or Savings
- The system automatically gives you an account number 
- Use the menu to interact with your account:
  - Deposit money 
  - Withdraw money
  - Pay monthly fee 
  - Check account status 
  - Close account 
  - Exit the program 
  -  If the account is closed, no actions can be performed

## Monthly Fee 

- Current account: $12
- Savings account: $20
- Each account can only pay the monthly fee once per program run 

## How to Run 

- Compile and run the program in your console:
  
```bash
javac contaBancaria.java
java contaBancaria
```
- Follow the instructions in the console to manage your account
